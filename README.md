### How do I load the Ghidra Project Archive (.gar) into Ghidra?

1. **Step 1:** Go to File and click Restore Project.

	<details>

	![Step 1 - Restore Project](misc/readme-assets/img-step1.png)
	</details>

2. **Step 2:** Select your archive file and choose the newest version (currently 1.1).

	<details>
	<summary>Show Step 2 image</summary>

	![Step 2 - Select Archive Version](misc/readme-assets/img-step2.png)
	</details>

3. **Step 3:** You can now view 250+ bookmarked functions.

	<summary>Show Step 3 image</summary>

	![Step 3 - View Bookmarked Functions](misc/readme-assets/img-step3.png)
	</details>
